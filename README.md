# WhatsApp BOT - Kecerdasan Buatan
## Fitur
`1. ChatGPT-3.5-Turbo`,
`2. Google Bard`,
`3. SimSimi`,
`4. ChattyAI`

## Instalasi
### Dapatkan Kunci API
Daftar dan dapatkan kunci API: <a href="https://api.danitechno.com">https://api.danitechno.com</a>

### Edit ./config/mainConfig.json
```json
....
"api": {
    "url": "https://api.danitechno.com",
    "key": "kunci-rahasiamu"
  }
```

### Instal NodeJs
```bash
apt install nodejs
```

### Instal dependensi
#### Gunakan Npm
```bash
npm install
```
#### Gunakan Yarn
```bash
yarn install
```

### Jalankan server
```bash
npm run start
```
